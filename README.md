# AWS-Portfolio-CICD

🚀 Deployed My Portfolio Website using AWS S3 + GitHub Actions CI/CD

Excited to share that I successfully deployed my personal portfolio website using a fully automated CI/CD pipeline!

🔧 Tech Stack:
• GitHub Actions – Automated deployment pipeline
• Amazon S3 – Static website hosting
• IAM – Secure access management
• Git – Version control


⚙️ How it works:
1️⃣ I push code to the main branch in GitHub
2️⃣ GitHub Actions automatically triggers the workflow
3️⃣ The pipeline syncs the portfolio files to the S3 bucket
4️⃣ The website gets updated instantly without manual deployment
